Open Mic Sign Up

A GitHub Pages friendly open mic app with an admin host view, a QR-powered public sign up page, editable running order, and automatic time slot planning.

Default: sign up at 6:00 PM Default: show starts at 7:00 PM Default: 3 songs or 15 minutes Admin can edit anything

How this is built

This starter uses GitHub Pages for hosting and Firebase Firestore for the shared live data. That gives you a static front end with a real-time sign up list that works across phones and the admin laptop.

Important: GitHub Pages alone cannot share live signups between different devices. The shared list needs a backend. Firebase is the easiest path for this kind of app.

Paste your Firebase config in the code first, then deploy to GitHub Pages. After that, the host can create an event, generate a QR, and performers can sign up from their phones.
