Open Mic Sign Up

A GitHub Pages friendly open mic app with a host admin view, QR powered public sign up, editable running order, and automatic slot planning.

v1.1 upgrades

Poster view for printing or tablet display, better signup success feedback, duplicate name warnings, check in and now performing controls, and a more venue friendly host workflow.
